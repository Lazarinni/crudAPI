## Instruções de uso:

### Antes de testar o projeto, baixar as dependência com npm

```bash
npm install
```

### Para rodar o servidor, execute o arquivo app.js

```bash
node app.js
```

OBS: O formulário para testes irá se encontrar no caminho "http://localhost:3000/"
